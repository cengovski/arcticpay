# Arctic Pay DApp

## 🚀 Fast USDC Transfers with Circle CCTP v2 + Fast Transfer Attestation + Onchain Travel Rule Identity

**Production-ready Next.js 15 DApp** for Arctic Pay on ARC Testnet.

### Features
- **Fast CCTP Bridge**: Iris-powered soft finality (1000 confirmations) transfers in seconds
- **Circle Identity**: Onchain Travel Rule compliance (`source.identities`)
- **Gasless Paymaster**: 50 USDC/day limit
- **Arctic Theme**: Ice gradients, snow particles, glassmorphism UI
- **Circle Sandbox + ARC Testnet** ready

### Quick Start (for AI Agent or Developer)
1. Clone: `git clone https://github.com/cengovski/arcticpay.git && cd arcticpay && git checkout feature/fast-cctp-identity`
2. `cp .env.example .env.local`
3. Fill Circle API keys
4. `npm install && npm run dev`

### Tech Stack
- Next.js 15 App Router + TypeScript
- Tailwind + shadcn/ui
- Circle App Kit + CCTP v2 + Iris API
- Cloudflare Pages deploy-ready
